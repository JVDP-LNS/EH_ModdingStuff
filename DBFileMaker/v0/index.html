List of available ItemTypes
ItemType
Number Name Description
0 Undefined Unused
1 Component The main file for a component
2 Device Used in Ship or Component files, to give ships certain attributes
3 Weapon Used in Component files, to control shooting of ammunition
4 AmmunitionObsolete Used in Component files, it is simple and customizable based on available prefabs
5 DroneBay Used in Component files, to control the spawning and stats of drones
6 Ship The main file for a ship
7 Satellite The main file for a satellite
8 ShipBuild Each of this file makes a ship layout for use by AI or acquiring by the player
9 SatelliteBuild Each of this file makes a satellite layout for use by AI or acquiring by the player
10 Technology Adds a specified technology node to the assigned faction's tech tree
11 ComponentStats Used in Component files, to define the stats of the component
12 ComponentMod Used in Component files, to add a possbile modifier to the stats
13 Skill Unused in EH
14 Faction The main file for a faction
15 Quest A file for quests, can be chained together for long quests
16 Loot Used in Quest files, A set of components, ships, satellites, currency etc. which can be dsiplayed, given to or retreived from the player
18 Fleet Used in Quest files, This file defines a quest battle, controls what ships appear
19 Character Used in Quest files, The main file for a quest character
20 QuestItem Used in Quest and Loot files, The main file for a quest item
25 Ammunition Used in Component, it allows a lot of customization
26 VisualEffect Used in Ammunition(Advanced), Devices and Weapon files, to define purely visual effects
27 BulletPrefab Used in Ammunition(Advanced), defines the hitbox behaviour of ammo
28 BehaviorTree Used in Ship, defines a custom AI that controls the ship in the hands of Autopilot or computer
29 GameObjectPrefab Used in Device, to customize objects created by certain devices
30 CombatRules Used in Fleet, to customize the battlefield
31 ComponentStatUpgrade Unused in EH
32 StatUpgradeTemplate Unused in EH
33 FrontierLevel Unused in EH
34 FrontierCommonLevel Unused in EH
35 ComponentGroupTag Used in Component, to restrict number of placeable modules belonging to the group
100 ShipSettings Global modifiers for ship stats
101 GalaxySettings Global settings, to define Game Start conditions and certain features in the entire map
102 DatabaseSettings Mod ettings, to give the mod name, version etc
103 ExplorationSettings Global settings, to modify certain features in all explorations
104 FrontierSettings Unused in EH
105 ShipModSettings Global settings, to modify all Ship Perks (upgrades to ship in the shipyard)
106 SpecialEventSettings Global settings, to modify date-based events that occur in the game
107 SkillSettings Global settings, to modify the skills available in the Player Skill Tree
108 DebugSettings Global settings, to modify behaviour of the secret console in the main menu
109 CombatSettings Global settings, to modify default Ship AI and CombatRules
110 UiSettings Global settings, to modify the game UI colors
111 FactionsSettings Unused in EH
112 MusicPlaylist Global settings, to modify the random list of music
113 ResearchSetting Unused in EH
114 PvpSettings Unused in EH
115 FrontierNpcSettings Unused in EH
116 FrontierLevelSettings Unused in EH
117 LocalizationSettings Global settings, modify what Corrosive Damage is called
